## Responsive Design

### Mobile First Approach

- **Mobile**: 320px - 767px (single column, optimized touch targets)
- **Tablet**: 768px - 1024px (flexible layouts)
- **Desktop**: 1025px+ (multi-column, full features)

### Key Principles

- Touch targets: minimum 44×44px
- Spacing: consistent 4px/8px/16px baseline
- Typography: readable at any size
- Navigation: accessible and intuitive

---

## Accessibility

- WCAG 2.1 AA compliance target
- Semantic HTML structure
- Keyboard navigation support
- Color contrast: minimum 4.5:1 for text
- Screen reader friendly

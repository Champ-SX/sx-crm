## 🔄 When to Archive New Files

Move documents to archive when:
- ✅ Sprint or phase is complete
- ✅ Feature is fully implemented and in production
- ✅ Decision is final and won't change
- ✅ Document is >6 months old

Keep in ACTIVE/ when:
- 🚀 Work is still in progress
- 📝 Document is actively maintained
- 🔄 Feature is in development or about to be
